name: Outreach preference - form-first, not cold calling
description: User prefers warm outreach via prospect lead forms over cold calling, finds cold calling intimidating
type: user

User is uncomfortable with cold calling and prefers a form-first outreach approach: submit the prospect's own lead form, then follow up by phone after they've either responded slowly or not at all. This turns the call into a warm follow-up that also demonstrates the problem Monstr solves.

When suggesting sales/outreach strategies, lean toward indirect approaches that build warm context before any phone call.
