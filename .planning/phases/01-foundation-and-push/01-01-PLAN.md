       - "build": "tsc -b && vite build"
       - "preview": "vite preview"

    2. Install dependencies from app/ directory:
       ```
       npm install react@^19 react-dom@^19 @supabase/supabase-js @tanstack/react-query react-router-dom react-i18next i18next i18next-browser-languagedetector lucide-react
       npm install -D typescript @types/react @types/react-dom @types/node vite @vitejs/plugin-react tailwindcss @tailwindcss/vite vitest
       npm install @capacitor/core @capacitor/push-notifications @capacitor/app @capacitor/haptics @capacitor/splash-screen @capacitor/badge
       npm install -D @capacitor/cli
       ```
       Use --legacy-peer-deps if React 19 peer dependency conflicts arise.

    3. Create app/vite.config.ts:
       - Plugins: react(), tailwindcss() from @tailwindcss/vite
       - resolve.alias: "@" -> path.resolve(__dirname, "./src")
       - server.port: 5174

    5. Create app/tsconfig.app.json targeting ES2020, module ESNext, moduleResolution bundler, JSX react-jsx, strict true, include ["src"].

    6. Create app/index.html with root div, script src="/src/main.tsx" type="module", lang="no", viewport meta.

    7. Create app/src/main.tsx: Bare React 19 root render of App component. Use createRoot from react-dom/client. Named exports only.

    8. Create app/src/App.tsx: Simple component returning a div with "Monstr" text. Named export only.

    9. Create app/src/index.css: Start with `@import "tailwindcss";` (Tailwind v4 pattern). No tailwind.config.js file.

    10. Create app/src/vite-env.d.ts with triple-slash reference to vite/client.

  <action>
    1. Run shadcn/ui init from app/ directory:
       ```
       npx shadcn@latest init --defaults
       ```
       This will set up components.json, update index.css with CSS variables, create src/lib/utils.ts.
       If it prompts, choose: New York style, Zinc color, CSS variables yes.

    2. Add shadcn/ui components needed for MVP:
       ```
       npx shadcn@latest add button card badge separator scroll-area input
       ```

    3. Create app/capacitor.config.ts:
       ```typescript
       import type { CapacitorConfig } from "@capacitor/cli"

       const config: CapacitorConfig = {
         appId: "no.monstr.app",
         appName: "Monstr",
         webDir: "dist",
         plugins: {
           PushNotifications: {
             presentationOptions: ["badge", "sound", "alert"],
           },
           SplashScreen: {
             launchAutoHide: true,
             backgroundColor: "#111111",
           },
         },
       }

       export default config
       ```

    4. Initialize Capacitor (do NOT add ios platform yet -- that's Plan 05):
       ```
       cd app && npx cap init "Monstr" "no.monstr.app" --web-dir dist
       ```
       If cap init conflicts with existing capacitor.config.ts, skip the init and just verify the config file exists.

    5. Verify shadcn/ui works: Update App.tsx to import and render a Button component with text "Test". Run build to confirm no import errors.
  </action>
